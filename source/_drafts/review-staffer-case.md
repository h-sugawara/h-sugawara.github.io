## 作品概要

時は１９６０年代、世の中の性質を捻じ曲げる物質「マナ」を扱い、かつては奇跡や魔法と呼ばれた「マナ現象」を発現できる超能力者「ステッパー」が存在していた。そして、ロンドンでは、ステッパーを適切に管理するための機関としてマナ現象管理局――管理局――が設立されていた。その管理局勤めになった一般人のノートリック・ケースが、マナ現象による殺人事件の真実を次々と究明する一方で、世界の真相も明らかになっていく、超能力ミステリ推理アドベンチャーゲーム。

### 主要な登場人物

プレイアブルキャラクターの「ノートリック・ケース」に加え、マナ現象管理局ではマナ事件専任チームに所属する「パンドリア・レッドフィンズ」と「テナ・ヘルベネス」、「コーカサス・ブリアン」がメインキャラクターとして登場し、他にも専任チームにゆかりのある人物や、課長や管理官などネームドではない人物も登場する。また、ＣＡＳＥごとに事件に関与する人物も登場する。

#### ノートリック・ケース

本作のプレイアブルキャラクターで主人公。本編開始前は、米国ニューメキシコ州にあるアキンボ探偵事務所で働いており、とある任務で渡航中だったマナ現象管理局ロンドン支部捜査課長にヘッドハンティングされた。事件を簡単に解決して、周りから尊敬されるような推理小説の主人公になりたいと考えていたため、スカウトを受けて、海を渡ってきた。他の主要人物とは異なりステッパーではないものの、演算能力は群を抜いて優れている。

#### パンドリア・レッドフィンズ

マナ事件専任チームのチーム長。危険レベル「エスティエンヌ」のステッパー。スキルは「接触型記憶想起」で、自身が接触した物体から、視覚や聴覚、触覚など記憶に関する情報を読み取れる能力である。ただし、無生物からは視覚情報を得られない。かつてはそのスキルを活かしていくつもの難事件を解決していた捜査官だったが、とある出来事がきっかけで他人との接触を忌避するようになり、現在ではその優秀さは鳴りを潜めている。

#### テナ・ヘルベネス

マナ事件専任チームに所属するメンバーの一人で、容疑者の取り調べを担う。危険レベル「コルト」のステッパー。スキルは「精密性振動感知」で、自身の近くにある振動を精密に検知できる。その性質を利用して、容疑者の心拍測定結果を付加した陳述書を取り調べの際に作成している。元々は保護観察チームに在籍していたが、４年前に起こった事件をきっかけに精神面で不調をきたしたことから、マナ事件専任チームへと異動してきた。

#### コーカサス・ブリアン

マナ事件専任チームに所属するメンバーの一人で、現場の調査を担う。危険レベル「ボウイ」のステッパー。スキルは「経過性痕跡視野」で、指紋や足跡などの痕跡が、いつ、どのくらい経過したのかを視覚を通して認識できる。そのスキルを使って、痕跡写真と呼ばれる書類を作成している。他人のお願いを断ることはほとんどなく、今までに起こった事件に関わる被害者のアフターフォローを自ら率先して行っている奇特な人物である。

### ゲーム本編

ゲーム本編は、一般的なＡＤＶに見られるような全てのストーリーを見るまで続く形式ではなく、事件ごとにＣＡＳＥという単位で分割されている。それぞれのＣＡＳＥ単位では、ＡＤＶパートの画面でストーリーが進行していき、事件現場の調査パートと、犯人・証拠・動機などを特定する推理パートの２種類の画面を何度か挟む構成である。

#### ＣＡＳＥ選択画面

タイトル画面から「始める」を押下した時に表示される画面。前述したＣＡＳＥ１からＣＡＳＥ５までの本編５つと、サイドストーリー２つの計７つから任意に１つ選んでそれぞれのストーリーを開始できる。なお、ＣＡＳＥは番号が小さい順で時系列が並んでおり、ＳｉｄｅＳｔｏｒｙ１「再スタート」は本編の前日譚で、ＳｉｄｅＳｔｏｒｙ２「準備」は本編の後日譚となっている。

{% gallery review-staffer-case case_select.webp 'ＣＡＳＥ選択' 719x404 %}
ＣＡＳＥ１以外から見始めると物語の繋がりが分からなくなるので、オススメしない
{% endgallery %}

#### ＡＤＶパート画面

現時点で集めた各資料の確認、バックログ、ウインドウ表示状態切り替え、オートモード切り替え、スキップモード切り替え、クイックセーブ、セーブ画面表示、ロード画面表示、設定画面表示ができる。
ちなみに、ヘルプにはキーマッピング表がないが、「Tab」キーでバックログ、「H」キーでウインドウ表示状態切り替え、「A」キーでオートモード、「Ctrl」キーでスキップモード、「Q」キーでクイックセーブ、「S」キーでセーブ画面表示、「L」キーでロード画面表示、「ESC」キーで設定画面表示ができる。なお、バックログを閉じたり、表示状態を戻したりなど、キャンセル操作をする場合は、同じキーを押下すればできる。

{% gallery review-staffer-case adv_part.webp 'ＡＤＶパート' 719x404 %}
セーブの最中は各機能が使えなくなるため、スキップモードと非常に相性が悪い
{% endgallery %}

#### 調査パート画面

調査パートとして、ブリアンのスキルを使った痕跡調査とレッドフィンズのスキルを使った記憶調査の２種類が存在する。なお、ブリアンとレッドフィンズの両者が同じ現場にいる場合、痕跡調査完了後に痕跡写真を入手したタイミングと同時に記憶調査の結果である記憶調査書も入手でき、プレイヤーの手間を省くような仕組みになっている。

##### 痕跡調査

画面内に表示されている「残った手がかり」の数をもとに、画面内にカーソルを当てて緑色に変化する箇所を一つずつクリックして、全ての手がかりを明らかにする調査パート。終了後に痕跡写真の書類を１つ入手できる。

##### 記憶調査

痕跡調査と同様の仕組みで、こちらはカーソルを当てて赤色に変化する箇所をクリックして、全ての手がかりを明らかにする調査パート。終了後に記憶調査書の書類を１つ入手できる。

{% gallery review-staffer-case memory_survey.webp '記憶調査' 719x404 %}
赤色のカーペットだと同じ色に変化するため、パッと見ただけでは気付きにくい
{% endgallery %}

#### 推理パート画面

事件の真相を推理するパートで表示される画面。犯人を一発で明らかにする仕様ではなく、疑問点などを一つずつ特定していく仕様になっている。

##### チュートリアル

推理パートのチュートリアルはＣＡＳＥ１に含まれているが、通常の推理パート中でもヘルプ画面で操作方法を確認できる。

##### 手がかり照合

事件の捜査で入手した資料から手がかりとなるワードやフレーズを選択して、正解を確認する画面。質問に対して手がかりを１つ探すパターンと、手がかりを２つ探して照合するパターンの２種類が存在する。

{% gallery review-staffer-case clue_matching.webp '手がかり照合' 719x404 %}
プレイヤーは、入手した文書と会話から手がかりを探してピン止めするだけ
{% endgallery %}

##### 六角推理

事件の本質に迫る際に行われる推理画面。手がかり照合が１つから２つまでのワードやフレーズを選択するのに対して、六角推理は名前の通り６種類の資料から選択するグレードアップバージョンである。

### コメント

#### コストパフォーマンス

ＵＩはチープな作りだが、全キャラクターがフルボイスで実装されているため、プレイ中に音声だけを聴いて内容を理解することが簡単であり、それだけでもお得感が満載である。

#### ゲームバランス

ＡＤＶとしての基本機能は備えているが、頻繁に発生するオートセーブによって各機能が保存完了までは使えなくなるため、操作性は少し悪い。

#### 世界観

超能力と探偵モノの組み合わせで物珍しさはないものの、大半の人間にとって異物の存在であるステッパーが関わる事件を軸に登場人物たちのヒューマンドラマが描かれており、没入感は非常に高い。

#### ゲーム性

プレイヤーに手がかりを探させて、その解決は主人公に任せるという形にすることで、そこそこの難しさを保ちながらも犯人を想像させながらプレイさせる面白さや満足感も兼ね備えている。

#### 音楽

作品の世界観を壊さないＢＧＭが揃っていた。しかしながら、強烈に印象に残るようなＢＧＭもないため、可も不可もない、という評価に落ち着いた。
