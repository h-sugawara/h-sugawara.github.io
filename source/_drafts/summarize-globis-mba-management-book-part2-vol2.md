## 製品とマーケティング

### 製品特性

製品は、その構成要素を「コア」「形態」「付随機能」の３つに分解できる。
マーケティング戦略上、これらのうちどれが最重要となるかは、製品特性や競争環境により異なる。また、製品導入期にはコアや形態で差別化が可能でも、市場が発達して競合類似製品が出回ると、付随機能での差別化が重要になる。
そのため、リサーチ等によって、顧客が重要視している要素を見極めなければならない。

{% gallery summarize-globis-mba-management-book-part2-vol2 product_components.svg '製品の構成要素' 352x352 %}{% endgallery %}

#### 製品の類型

顧客の製品への関心の持ち方、買い方やKBFは、製品の性質により異なる。そのため、製品の特徴によっていくつかのカテゴリー分けて把握すると、効果的なマーケティング戦略を策定する時の早道になる。実務で頻出する製品分類は、物理特性・使用目的・顧客の購買プロセスという切り口で分けたものである。

{% gallery summarize-globis-mba-management-book-part2-vol2 product_classification.svg '製品の分類' 633x642 %}{% endgallery %}

### ブランド

ブランドとは、「競合相手やその製品・サービスとの違いを明確にするために用いられる名前・言葉・デザイン・シンボル、または、それらを組み合わせたもの」で、差別化の核となる。特に、製品自体に大差が無い場合や機能面等の差別化に多くの時間・コストを要する場合等は、ブランドが大きな武器となる。

{% gallery summarize-globis-mba-management-book-part2-vol2 brand_functions.svg 'ブランドの機能' 572x362 %}{% endgallery %}

#### 強いブランドの効果

強いブランドを育て、ブランド・ロイヤルティを高めることは、重要な戦略課題である。ブランドの価値は、一朝一夕に高められるものではなく、一貫したコミュニケーション活動はもちろん、高い品質・サービスの維持、行き届いたアフターサービスの提供、環境保全への取り組み、地域社会への貢献等、地道な努力の積み重ねが必要になる。
企業は、強いブランドを持つことで、下記のような効果を得ることができる。

{% gallery summarize-globis-mba-management-book-part2-vol2 strong_brand_power.svg '強いブランドの効果' 523x623 %}{% endgallery %}

#### ブランド展開と体系

ブランドの展開にはいくつかの方法がある。例えば、コーポレート・ブランドと製品ブランドを組み合わせるやり方は、日本の顧客が品質保証機能としてコーポレート・ブランドを重視する傾向があるため、良くとられる。また、１つの製品カテゴリで複数製品を同時販売する場合、全製品に異なるブランドをつける方法と、中心的なブランドを複数製品に冠する方法がある。
ブランド展開を検討する際は、ターゲット顧客・製品とブランドイメージとの整合性、マーケティングコスト・自社製品間のポジショニング・シナジー・リスク等を十分に考慮しなくてはならない。

{% gallery summarize-globis-mba-management-book-part2-vol2 brand_system.svg 'ブランド体系' 506x312 %}{% endgallery %}

### 新製品開発

新製品開発プロセスは、下記の４つのステージに大別できる。

{% gallery summarize-globis-mba-management-book-part2-vol2 new_product_development_process.svg '新製品開発プロセス' 601x470 %}{% endgallery %}

#### ニーズ発想とシーズ発想

新製品のアプローチには、ニーズ型開発とシーズ型開発の２つがある。前者は「とあるニーズを解決する方法は無いか？」という発想から開発を始めるのに対し、後者は「開発した新しい技術を何かに利用できないか？」という発想で取り組む。
新製品や新規事業を成功させるためには、顧客ニーズを無視できない。そのため、ニーズ発想の重要性が強調されることが多いが、新市場を創造する場合等は最初からニーズがはっきりしているとは限らず、シーズ発想によるアイディア開発が有効な場合も多い。また、ニーズを発見しても、技術的解決できなければ新製品は完成しない。
現実の製品開発過程では、状況に応じてニーズ型とシーズ型を両立させ、合致させることが望ましい。

#### 製品アイディアと製品コンセプト

新製品開発の出発点は、市場に提供するその製品固有の機能である「製品アイディア」を考えることである。この段階では、企業は経営資源を駆使して、出来るだけ多くのアイディアを創出する。そのためにも、企業はシステマチックにアイディアを出し続ける仕組みを整備することが重要である。
最近では、社内に広くアイディアを募ったり、社内ベンチャーを奨励する企業も増えている。このような手法は、社員の問題意識を高め、士気高揚や組織活性化につながる場合もある。
製品アイディアは、企業理念・戦略領域・経営資源・経済性・市場性・実現性等の観点から検討され、成功確率の高いアイディアを絞り込み、開発の優先順位を付ける。このプロセスを、アイディアのスクリーニングと言う。
スクリーニングを経たアイディアは、誰にどのような価値を提供するか、を念頭に置きながら、明確かつ詳細な言葉に落とし込む。こうして、想定するユーザーが、実際にそれを使っている場面を想像出来るまでに具体化されたものを製品コンセプトと言う。
製品コンセプトは、「誰がこのコンセプトに共感するか」「競合製品に対する優位性は何か」「従来の製品・サービスに比べて改良点は何か」「値段はどうするか」「購入決定者は誰か」「使うのは誰か」等の詳細を問うコンセプト・テストを通して絞り込んでいく。
