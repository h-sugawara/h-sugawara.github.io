2025年8月7日に Steam と Nintendo Switch で発売された、ロジックパズルゲーム「Is This Seat Taken?」をレビューします。

<!-- more -->

## 作品概要



### メイン画面

#### マップ・ステージ選択画面

プレイできるマップは、「バルセロナ（スペイン）」「ブリュッセル（ベルギー）」「ロンドン（イングランド）」「ニューヨーク（アメリカ）」の４都市計５つある（注：バルセロナのマップのみ２種類存在する）が、最初に選択できるのはバルセロナマップのみとなる。
それぞれのマップごとに６つのステージがあり、プレイヤーが最初から全てのステージを選ぶことはできず、左のステージからルートに沿って右へと順番に５つのステージをクリアする流れとなる。それらをクリアすると、６つ目の隠しステージと、次の都市のマップがアンロックされる（例：バルセロナマップをクリアするとブリュッセルマップが解放される）。

## 作品評定

### 評点

{% message color:success %}
77点（良/B+ランク）
{% endmessage %}

数日程度の余暇時間を過ごすために、一風変わったロジックパズルゲームを、お安い価格で買ってプレイしたい人にオススメしたいゲーム。

#### 評点の内訳

{% message %}
美少女ゲームと採点基準が異なることに注意してください。
{% endmessage %}

### コメント

#### コストパフォーマンス

ロープライス帯のゲームとしては高品質ではあるが、実績を全て達成する以外には、やり込み要素やエンドコンテンツ要素がないため、お得感はそこそこ。

#### ゲームバランス

説明なしでも直感的にプレイできてストレスは感じにくいが、フォルムンが前後に被っている時に、入れ替えと配置のどちらのモードか分かりにくく、操作に手間取った点が残念。

#### 世界観

学校の教室や飲食店、待合室など、特定の空間内にある座席に誰を座わらせるか、人間関係を考慮して座席決めをすることを、ロジックパズルとして落とし込み、上手く表現できている。

#### ゲーム性

ジャンルとして完成されたパズルゲームの要素そのものの楽しさに寄りかかっており、このゲーム単体の要素に絞ると、プレイヤーを引き込むほどのストーリー性がなく、面白みには欠ける。

#### 音楽

ＢＧＭにもう少し力を入れても良かったのではないか。プレイ中に何が流れているか分からないレベルのため、大減点とした。
